# Hadoop Demo

1. 下载 hadoop-2.9.1.tar.gz
    ```
    wget http://mirrors.hust.edu.cn/apache/hadoop/common/hadoop-2.9.1/hadoop-2.9.1.tar.gz
    ```

2. 启动 docker 容器

   ```
   docker-compose up -d
   ```

3. 访问以下网址确保成功

   ```
   # hadoop
   http://localhost:50070 
   # yarn
   http://localhost:8088 
   ```
